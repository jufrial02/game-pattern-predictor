# Game Pattern Predictor Tools

Aplikasi berbasis Python untuk membaca riwayat (history) pada game angka/warna/RNG menggunakan visual scanning, menganalisis probabilitas kemunculan pola secara matematis, dan menampilkan prediksi hasil berikutnya secara real-time.

## Struktur Proyek
- `main.py`: File utama untuk menjalankan tools.
- `modules/scanner.py`: Modul untuk membaca data/warna dari layar.
- `modules/analyzer.py`: Modul otak yang menghitung rumus probabilitas pola.
- `modules/overlay.py`: Modul untuk menampilkan UI prediksi melayang di layar.
